# Handwriting Appraisal & Character Recognition (筆跡鑑定・文字認識AI)

## 概要 (Overview)
このリポジトリは、高校（情報工学科）の**課題研究**としてゼロから企画・開発した「機械学習を用いた筆跡鑑定・二要素認証システム」のプロジェクトです。
現在、過去の開発環境からGitHubへの移行（引っ越し）作業中のため、ファイル構成やドキュメントを随時アップデートしています（Work in Progress）。

単なるAIモデルの学習にとどまらず、学習したモデルを実際に活用するためのTkinterを用いたGUIアプリケーション、さらには独自のアルゴリズムによるパスワード認証機能を組み合わせた「二要素認証（画像認識＋パスワード）」システムとして一貫して実装している点が特徴です。

## プロジェクトの特徴 (Features)

### 1. ディープラーニングによる文字・筆跡認識モデルの構築
* **EMNISTデータセットの活用:** 拡張MNIST（EMNIST）データセットを用いて、英数字を高精度に分類するCNN（畳み込みニューラルネットワーク）モデルを構築。
* **独自の画像データ学習:** OpenCV (`cv2`) を利用して手書き文字の画像データを前処理（リサイズ・グレースケール化）し、特定の筆跡を鑑定するモデルを実装。
* **強力なデータ拡張 (Data Augmentation):** アフィン変換（回転、平行移動、拡大縮小）を網羅的に適用する独自のスクリプトにより、訓練データを大幅に水増ししてモデルの汎化性能を向上。
* **誤認識の可視化:** テストデータに対してAIが間違えて推論した画像を抽出し、どこを間違えたのかをmatplotlibで視覚的に分析する機能を搭載。

### 2. TkinterベースのGUIアプリケーションと二要素認証
* **直感的な操作画面:** ユーザーが鑑定したい筆跡画像を選択し、ボタン一つでAIによる判定結果を確認できるデスクトップアプリを実装。
* **セキュアなアクセス制御:** 鑑定結果が特定個人の筆跡だった場合、その後の操作に制限をかける「パスワード認証用モーダルダイアログ」が起動。規定回数パスワードを間違えるとシステムにロックがかかるセキュリティ機能を実装。
* **独自のパスワード暗号化処理:** 入力された文字列をASCIIコードに変換し、文字列長や合計値を用いた独自の計算ロジックによってパスワードの照合を行うセキュアな設計。

## ディレクトリとファイル構成 (Directory Structure)
機能ごとにディレクトリを分割して管理しています。

### 📁 `docs/` (ドキュメント・資料)
* `presentation_slides.pdf`: 課題研究発表時に使用した、本システムの目的や理論、二要素認証の仕組みを図解したプレゼンテーション資料です。

### 📁 `src_train/` (AIモデル学習・評価用プログラム)
* `emnist_test.py`: EMNISTデータセットの読み込み、データ拡張、CNNを用いた学習と評価を行うメインスクリプト。
* `handwrite_test.py`: 独自の手書き画像データを読み込み、筆跡鑑定モデルの学習を行うスクリプト。
* `mk_dataset.py`: ディレクトリ内の画像群からデータセット（配列）を生成する前処理スクリプト。
* `mk_data.py`: アフィン変換を用いてデータセットの枚数を数学的に増幅させるデータ拡張スクリプト。
* `list_get.py`: 学習過程のログから損失関数 (Loss) と正解率 (Accuracy) の推移を抽出するユーティリティ。
* `model_test.py`: 学習済みモデルのテスト評価を実行するスクリプト。
* `misclassified_mnist.py`: モデルの誤認識画像一覧を抽出・描画する分析ツール。

### 📁 `src_gui/` (筆跡鑑定GUIアプリケーション)
* `tk_GUI.py` / `GUI.py`: アプリケーションのメイン画面（UI構築とイベント処理）を担うスクリプト。
* `num_test01.py`: GUIから渡された画像を前処理し、学習済みモデルを用いて推論（筆跡の判定）を行うバックエンド処理モジュール。
* `create_modal_dialog.py`: 筆跡判定後に起動するパスワード入力用のモーダルダイアログ（ポップアップ画面）を制御するスクリプト。
* `processing.py`: ユーザーが入力したパスワード文字列を独自のASCIIアルゴリズムで暗号化・変換するモジュール。
* `encode.py`: パスワード認証用のマスターデータ（ハッシュ値のようなもの）を生成し、テキストファイルに出力する事前準備用スクリプト。

## 技術スタック (Tech Stack)
* **言語:** Python 3.x
* **AI/機械学習:** TensorFlow, Keras, scikit-learn
* **画像処理:** OpenCV (`cv2`), Pillow (`PIL`)
* **数値計算・可視化:** NumPy, Matplotlib
* **GUI開発:** Tkinter

## 注意事項 (Note)
* **学習済みモデルについて:** `.h5` 形式の学習済みモデルファイルは容量が大きいため、Gitリポジトリの管理対象から除外しています。動作させる場合は、`src_train/` 内の学習スクリプトを実行してモデルを再生成してください。
