# character_recognition

## 概要 (Overview)
このリポジトリは、高校の**課題研究**として作成した文字認識AIのプロジェクトです。
現在、過去の開発環境からGitHubへの移行（引っ越し）作業中のため、ファイル構成やドキュメントは完全な状態ではなく、随時アップデートを行っています（Work in Progress）。

## プロジェクトの特徴 (Features)
TensorFlowおよびKerasを使用し、文字画像の認識モデルを構築・評価するためのスクリプト群が含まれています。

* **EMNISTデータセットの活用:** EMNIST（拡張MNIST）データセットを用いて、英数字の分類を行うモデルを構築しています。
* **独自の画像データ学習:** 手書き文字の画像データ（OpenCVでリサイズ・グレースケール化）を読み込み、学習させる機能も実装しています。
* **強力なデータ拡張 (Data Augmentation):** アフィン変換（回転、平行移動、拡大縮小）を用いて訓練データを大幅に水増しし、モデルの汎化性能を向上させる処理を組み込んでいます。
* **誤認識の可視化:** テストデータに対してモデルが間違えて推論した画像を抽出し、どこを間違えたのかを視覚的に分析する機能を持っています。

## ファイル構成 (File Description)
現在リポジトリに含まれている主要なファイルとその役割は以下の通りです。

### 1. メインの学習プログラム
* `emnist_test.py`: EMNISTデータセットを読み込み、アフィン変換によるデータ拡張を行った上で、CNN（畳み込みニューラルネットワーク）を用いて文字認識モデルの学習と評価を行うメインスクリプトです。
* `handwrite_test.py`: 自身で用意した手書き画像データを読み込み、学習・評価を行うためのスクリプトです。

### 2. データセット・前処理関連
* `mk_dataset.py`: 指定したディレクトリから画像データを読み込み、OpenCV (`cv2`) を使ってグレースケール化と指定サイズへのリサイズを行い、データセットとして出力します。
* `mk_data.py`: 学習用の画像配列を受け取り、アフィン変換（回転・移動・スケール変更）を網羅的に適用してデータ数を劇的に増やすスクリプトです。

### 3. 評価・分析関連
* `model_test.py`: 保存された学習済みモデル (`emnist_model.h5`) を読み込み、テストデータに対する損失関数 (Loss) と正解率 (Accuracy) を計算してファイルに出力します。
* `list_get.py`: 学習過程を記録したテキストファイルから、特定の行を抽出して損失関数や正解率の推移データを取得するユーティリティです。
* `misclassified_mnist.py`: 学習済みモデルを使用してテストを行い、推論結果と正解ラベルが一致しなかった画像（誤認識した画像）をmatplotlibで並べて表示します。

## 技術スタック (Tech Stack)
* Python
* TensorFlow / Keras (ディープラーニングフレームワーク)
* NumPy (数値計算)
* Matplotlib (グラフおよび画像の描画)
* OpenCV (`cv2`) (画像処理)
* scikit-learn (交差検証などのデータ分割)
